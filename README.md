## 购买注意事项

几个实际买的时候容易踩的坑：

**注册时关掉代理。** 搬瓦工对注册 IP 比较敏感，挂着梯子注册容易被判定为欺诈然后自动退款。注册和下单时用真实网络环境。

**填真实信息。** 不用担心隐私，但地址信息看起来一眼假的话，同样有可能被系统砍单。

**官网主域名可能被 DNS 污染。** 主站 bandwagonhost.com 在国内有时解析有问题，可以通过官方镜像 bwh81.net 或 bwh89.net 访问，功能完全相同。

**限量版套餐抢到就是赚到。** 搬瓦工时不时会推出限量版套餐，同等配置价格往往是常规套餐的三分之一到五分之一，但库存少，上架几小时内就可能卖完。有需要可以关注补货通知渠道。
